Word Guess Game

The word guess game I designed is a play on the classic game of Hangman. Players will be given an image then will guess letters to fill in the word. Users have 6 tries before receiving a new image to guess. The technologies used were HTMl, CSS, and Javascript.

Crystal Collector

The crystal collector game was designed using jQuery. Users see 6 crystal options, each of which has a randomly assigned numerical value. The game then generates a random number that players need to match by clicking on the crystals. The page updates to show the number they need to match, along with what they're current score is.
